# Posts Instagram - NIDUS Ventures

## Estrutura

- `posts-data.json` - Dados de todos os posts
- `logo-nidus.png` - Logo da NIDUS em PNG (transparente)
- `post-XX-*.html` - Templates HTML para cada post (1080x1080px)

## Como Usar

1. Abra o arquivo HTML do post no navegador
2. Tire screenshot ou use ferramenta de captura
3. Salve como PNG/JPG 1080x1080px
4. Poste no Instagram

## Tamanhos

- **Feed**: 1080x1080px (quadrado)
- **Stories**: 1080x1920px (vertical)
- **Reels**: 1080x1920px (vertical)
